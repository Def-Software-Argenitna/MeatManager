👥 Gestión de Clientes

Registrar y administrar datos de clientes

← Volver al Índice

¿Por Qué Registrar Clientes?

La gestión de clientes permite:

  • Conocer quién compra y cuánto gasta
  • Ofrecer promociones personalizadas
  • Gestionar créditos y deudas
  • Generar reportes por cliente
  • Mejorar el servicio y retención
  • Crear perfiles para entregas a domicilio

Acceder a Clientes

1 Desde el menú lateral: Busca la opción Clientes
2 Se abrirá una tabla con todos tus clientes registrados

Crear un Nuevo Cliente

1 Haz clic en el botón "+ Nuevo Cliente"
2 Completa los datos:
Campo Descripción Requerido
Nombre Nombre completo o razón social ✓
CUIT/CUIL/DNI Identificación fiscal o personal Recomendado
Email Correo electrónico del cliente Recomendado
Teléfono Número de contacto principal Recomendado
Dirección Dirección completa Recomendado
Localidad Ciudad o zona Recomendado
Condición IVA Responsable Inscripto, Consumidor Final, etc. Recomendado
3 Haz clic en Guardar
✓ El cliente se creará automáticamente

Editar Datos de un Cliente

1 Busca el cliente en la tabla
2 Haz clic en el botón "Editar" o en la fila
3 Modifica los datos que necesites cambiar
4 Haz clic en Guardar

Búsqueda de Clientes

Usa la barra de búsqueda para encontrar clientes rápidamente:

  • Escribe el nombre del cliente
  • O el CUIT/CUIL/DNI
  • O el teléfono
  • El sistema filtrará en tiempo real

Historial de Compras por Cliente

Desde la pantalla de clientes, puedes:

  1. Seleccionar un cliente
  2. Ver "Historial de Ventas"
  3. Consultar montos gastados
  4. Ver fechas de últimas compras
  5. Identificar clientes VIP o inactivos

Eliminar un Cliente

Solo se pueden eliminar clientes sin historial de compras.

1 Busca el cliente que deseas eliminar
2 Verifica que no tenga compras registradas
3 Haz clic en "Eliminar"
4 Confirma la eliminación

Tipos de Clientes

Tipo Descripción Datos Importantes
Consumidor Final Comprador ocasional o regular Nombre, teléfono (para referencias)
Negocio/Restaurante Cliente comercial CUIT, razón social, contacto, dirección, cond. IVA
Delivery/Plataformas Clientes de aplicaciones Dirección para entrega, teléfono de contacto
Multitienda Cadenas o reventa CUIT, múltiples direcciones, términos de pago
