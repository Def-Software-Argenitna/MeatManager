💡 Consejos Rápidos

  • Código de Barras: Los códigos Systel de las balanzas se leen automáticamente en Ventas
  • Integración QENDRA: Si tenés balanza Systel, usá el botón 🎫 Tickets en Ventas para cobrar directamente desde la balanza
  • Producto-Stock-Precio: Crear un producto automáticamente genera entrada de Stock y Precio
  • Búsqueda Rápida: En todas las páginas puedes buscar por nombre, código PLU o ID
  • Descarga de Datos: Los reportes pueden exportarse para análisis en Excel
  • Guardado Automático: Los cambios se guardan automáticamente en la base de datos
