⚖️ Balanza Real y Modo Test

Conexion correcta, uso en ventas y despostada, y como salir del modo test

Inicio / Balanza Real y Modo Test

✅ Requisitos basicos

  • Balanza Systel Cuora Max conectada por USB.
  • PC con Windows y permisos para seleccionar el puerto COM.
  • MeatManager PRO instalado y abierto.

1) Como conectar la balanza real

1 Conecta la balanza por USB y asegurate de que este encendida.
2 En Ventas o en cualquier despostada, presiona Conectar Balanza.
3 Selecciona el puerto (COM) que te muestra Windows y acepta.
4 Si la conexion es correcta, veras el estado Balanza OK.
💡 Consejo: si te aparece mas de un COM, proba el que se agrego al conectar la balanza.

2) Importar productos desde la balanza (Stock e Inventario)

1 Anda a Stock e Inventario.
2 Presiona Importar de Balanza.
3 El sistema traera nombres, PLUs y precios cargados en la balanza.
✅ Esto sirve para sincronizar articulos sin cargarlos a mano.

3) Que es el Modo Test y para que sirve

El Modo Test simula la balanza: genera pesos falsos para poder probar ventas o despostada sin la balanza conectada.

  • Ideal para capacitacion y pruebas.
  • No requiere conexion USB.
  • Usa pesos aleatorios (no reales).
⚠️ Importante: cuando esta activo, la balanza real queda desactivada.

4) Como activar y salir del Modo Test

1 Presiona (Probar) / Modo Test.
2 Aparecera un aviso explicando que el modo test desactiva la balanza real. Confirma si queres continuar.
3 Cuando esta activo veras un cartel: Modo Test activo.
4 Para salir, presiona Salir de Modo Test y confirma.

5) Si no toma la balanza real

  • Verifica que el Modo Test este desactivado.
  • Revisa el cable USB y que la balanza este encendida.
  • Proba otro puerto COM.
  • Cerra y vuelve a abrir MeatManager.
