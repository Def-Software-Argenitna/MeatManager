¿Qué es Productos de Compra?

"Productos de Compra" es el catálogo central donde defines todos los productos que vendes. Es la única entrada para crear productos en el sistema.

Cuando creas un producto aquí, el sistema automáticamente:

  • ✓ Crea un registro en Stock (cantidad inicialmente en cero)
  • ✓ Crea un registro en Precios de Venta (con el precio que indiques)
  • ✓ Asigna el código PLU para lectora de códigos
  • ✓ Lo hace disponible en Compras para registrar entradas
  • ✓ Lo hace disponible en Ventas para realizar transacciones

Flujo Correcto de Productos

1. Crear Categoría (ej: "Carnes Rojas") ↓ 2. Crear Producto en "Productos de Compra" ↓ 3. Automáticamente se genera: - Entrada en Stock (cantidad = 0) - Entrada en Precios de Venta ↓ 4. Registrar Compra → Stock aumenta ↓ 5. Realizar Venta → Stock disminuye
✓ Este flujo garantiza que todos los productos tengan información completa

Acceder a Productos de Compra

1 Desde el menú lateral: Busca Productos de Compra
2 Se abrirá una tabla con todos los productos creados

Crear un Nuevo Producto

1 Haz clic en "+ Nuevo Producto"
2 Completa la información BÁSICA:
Campo Descripción Requerido
Nombre Nombre del producto (ej: "Carne Vaca - Asado") ✓
Categoría Selecciona una categoría existente ✓
Unidad kg, unidad, litro, etc. ✓
Descripción Detalles adicionales (opcional)
3 Completa la información de VENTAS:
Campo Descripción Requerido
Código PLU Código de 5 dígitos para la balanza Systel ✓
Categoría de Venta Categoría para reportes de ventas (puede diferir de la de compra) ✓
Precio de Venta Precio unitario para vender (ej: $250/kg) ✓
4 Haz clic en Guardar
✓ El producto se creará y automáticamente généarán entradas en Stock y Precios

Información Importante sobre PLU

ℹ️ Código PLU (Price Look-Up): Es un código de 5 dígitos usado por las balanzas Systel. Ejemplos:
Producto PLU Recomendado Razón
Carne Vaca - Asado 10001 Primer producto de carnes rojas
Pollo Entero 20001 Primer producto de aves
Cerdo - Milanesa 30001 Primer producto de cerdo

Tip: Usa rangos de 5 dígitos por categoría para organizarte mejor.

Editar un Producto Existente

1 Busca el producto en la tabla
2 Haz clic en "Editar" o en la fila
3 Modifica los datos que desees cambiar
4 Haz clic en Guardar cambios
✓ Los cambios se aplicarán automáticamente en Stock y Precios

Eliminar un Producto

ℹ️ Nota: Solo se pueden eliminar productos que NO tengan stock o compras registradas. Se recomienda mantener todos los productos por historial.

Búsqueda y Filtrado

Usa la barra de búsqueda para encontrar productos rápidamente:

  • Por Nombre del producto
  • Por Código PLU
  • Por Categoría
  • La búsqueda es en tiempo real
