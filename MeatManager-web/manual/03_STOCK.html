¿Qué es el Stock?

El stock es la cantidad disponible de cada producto en tu carnicería. La gestión correcta del stock permite:

  • Saber qué productos tienes disponibles
  • Realizar ajustes manuales (pérdidas, deterioros, etc.)
  • Evitar sobrevender productos
  • Generar el precio de venta según cantidad
  • Planificar compras futuras

Acceder a Stock

1 Desde el menú lateral: Busca la opción Stock
2 Se abrirá una tabla con todos tus productos y su cantidad disponible
✓ El stock se actualiza automáticamente al registrar compras o ventas

Entender la Interfaz

En la página de Stock verás:

Columna Descripción
Producto Nombre del producto
Categoría A qué categoría pertenece
Cantidad Stock actual en kg (o unidad)
Acciones Botones para ajustar el stock

Realizar un Ajuste de Stock

Usa ajustes manuales para corregir diferencias de inventario por:

  • Productos deteriorados o perdidos
  • Errores en la carga de compras
  • Inventario físico contra sistema
  • Muestras o regalos

Paso a Paso:

1 Busca el producto en la tabla (usa el buscador si hay muchos)
2 Haz clic en el botón "Ajustar" o el icono de engranaje
3 Se abrirá un diálogo con opciones de ajuste
4 Selecciona si sumar o restar stock (+ o -)
5 Ingresa la cantidad a ajustar (ej: 2.5 kg)
6 Haz clic en Confirmar
✓ El stock se actualizará automáticamente

Tipos de Ajuste

Tipo Símbolo Cuándo Usar
Sumar (+) ➕ Cuando agregas stock (compra urgente, devolución, etc.)
Restar (-) ➖ Cuando pierdes stock (deterioro, regalo, error, etc.)

Cómo se Actualiza el Stock Automáticamente

El sistema ajusta el stock automáticamente en estos casos:

Acción Efecto en Stock
Registrar una compra Suma la cantidad comprada
Registrar una venta Resta la cantidad vendida
Crear producto desde Productos de Compra Crea entrada con stock = 0

Búsqueda de Productos

Si tienes muchos productos, usa la barra de búsqueda:

  • Escribe el nombre del producto
  • O escribe el código PLU
  • O escribe el ID del producto
  • El sistema filtrará en tiempo real
✓ La búsqueda es sensible a mayúsculas/minúsculas

Stock en Cero

⚠️ Importante: Si un producto tiene stock = 0, no aparecerá disponible para vender. Si intentas vender un producto sin stock, el sistema te avisará.

Para reactivar un producto:

  1. Ajusta el stock agregando cantidad (+)
  2. O registra una compra del producto
  3. El producto volverá a estar disponible

Historial de Movimientos

ℹ️ El sistema guarda automáticamente un historial de todos los movimientos de stock. Puedes consultarlo en Informes PRO para análisis detallado.
