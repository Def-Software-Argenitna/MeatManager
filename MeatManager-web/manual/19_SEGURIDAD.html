🔒 Seguridad y Usuarios

Control de accesos y permisos dentro del sistema

← Volver al Indice

Objetivo

Definir usuarios y permisos para evitar cambios no autorizados:

  • Controlar quien puede vender, comprar o ajustar stock
  • Proteger reportes e informacion sensible
  • Evitar errores accidentales

Crear un Usuario

1 Ingresa a Seguridad
2 Haz clic en + Nuevo Usuario
3 Completa nombre, usuario y contrasena
4 Asigna el rol o permisos
5 Guarda el usuario

Roles Sugeridos

Rol Permisos
Administrador Acceso total a configuracion, reportes y ajustes
Encargado Ventas, compras, stock, cierres de caja
Cajero Solo ventas y consultas basicas
Operario Solo despostada y registros internos

Buenas Practicas

  • Usa contrasenas unicas por usuario
  • Cambia contrasenas cada cierto tiempo
  • Desactiva usuarios que ya no trabajan
  • Limita permisos al minimo necesario
Si el sistema no usa usuarios internos, gestiona el acceso desde Windows.
