¿Qué es Cierre de Caja?

El cierre de caja es el proceso de finalizar el día de ventas. Permite:

  • Contabilizar todas las ventas del día
  • Verificar que money matches (dinero coincida con ventas)
  • Generar reportes diarios
  • Identificar diferencias o faltantes
  • Crear un registro histórico de transacciones
  • Preparar depósitos bancarios

¿Cuándo Cerrar Caja?

Idealmente: Al final del día de atención

Frecuencia recomendada: Diaria

Horario sugerido: Después de las últimas ventas

✓ Los cierres diarios ayudan a identificar problemas rápidamente

Acceder a Cierre de Caja

1 Desde el menú lateral: Busca Cierre de Caja
2 Se abrirá la interfaz con información del día

Paso a Paso: Realizar un Cierre

1 Verifica que NO haya ventas pendientes de registrar
2 Consulla el TOTAL DE VENTAS mostrado en la pantalla
3 Desglose por forma de pago:
  • Cuánto en Efectivo
  • Cuánto en Tarjeta
  • Cuánto en Transferencia
  • Otros medios
4 Si existe, ingresa monto en caja física (efectivo contado manualmente)
5 El sistema calcula automáticamente: Diferencia = Monto Real - Monto Sistema
6 Si hay diferencia, especifica:
  • ¿Es faltante o sobrante?
  • ¿Probable causa?
  • Notas adicionales
7 Haz clic en Cerrar Caja
✓ El cierre se registrará como finalizado

Información que Genera el Sistema

Dato Descripción
Fecha/Hora Cierre Cuándo se realizó el cierre
Total Ventas Suma de todas las ventas del período
Cantidad Transacciones Cantidad de ventas registradas
Detalle por Medio de Pago Desglose de efectivo, tarjeta, transferencia, etc.
Diferencia si la hay Faltante o sobrante detectado

Diferencias Comunes y Causas

Situación Probable Causa Acción
Faltante pequeño (< 5%) Error de vuelto, transferencia pendiente Documentar y monitorear
Faltante grande Venta no registrada, robo Revisar historial de ventas
Sobrante Cliente pagó extra, error de sistema Investigar y retornar si aplica
Gran diferencia Errores no capturados en el día Revisar todas las ventas

Reapertura de Caja (Editar Cierre)

⚠️ Cautela: Solo usa esto si necesitas corregir algo del mismo día. Regularmente no se debe reabrir.
  1. Desde el historial de cierres, selecciona el cierre a editar
  2. Haz clic en "Reabrir"
  3. Realiza las correcciones necesarias
  4. Vuelve a cerrar

Historial de Cierres

El sistema mantiene un registro completo de todo los cierres realizado:

  • Puedes consultar información histórica
  • Filtrar por rango de fechas
  • Generar reportes consolidados
  • Exportar datos para auditoría

Impacto en Otros Módulos

Módulo Efecto
Ventas Las ventas pueden seguir registrándose después del cierre
Informes PRO Los datos de cierre se incluyen en reportes diarios
Stock Ya se actualizó durante las ventas, no se ve afectado
