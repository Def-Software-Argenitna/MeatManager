1) Inicio y configuración base

  1. Ingresar con usuario administrador y validar sucursal activa.
  2. Revisar productos, medios de pago y permisos antes de abrir caja.
  3. Si hay balanza, sincronizar artículos y precios antes de vender.
  4. Realizar venta de prueba para validar ticket, stock y caja.
Captura real pendiente: portada del sistema con sucursal activa.

2) Productos, categorías y stock

  1. Dar de alta artículos con nombre comercial, unidad, categoría y PLU.
  2. Normalizar categorías para evitar duplicados.
  3. Usar ajustes con motivo para trazabilidad.
  4. Confirmar precios vigentes antes de cambios masivos.
Captura real pendiente: módulo Stock e Inventario.

3) Compras y proveedores

  1. Registrar proveedor con datos fiscales y condiciones.
  2. Cargar compra con cantidades y costos.
  3. Validar comprobantes y comparar históricos.
  4. Normalizar productos de compra.
Captura real pendiente: módulo Compras (cabecera + ítems).

4) Ventas, promociones y balanza

  1. Buscar por nombre, PLU o código de barra.
  2. Aplicar promociones por kg según configuración.
  3. Cobrar por medio simple o pago dividido.
  4. Sincronizar PLU/precios de balanza y validar tickets.
  5. Usar eliminación de ticket con control de seguridad.
Captura real pendiente: pantalla de Ventas (carrito + cobro).
Captura real pendiente: integración de balanza (sync/config).

5) Pedidos y clientes

  1. Registrar clientes con datos de contacto.
  2. Tomar pedidos manuales o desde canal digital.
  3. Asignar estado operativo.
  4. Conservar historial para seguimiento.
Captura real pendiente: módulo Pedidos.

6) Caja y cierre diario

  1. Registrar apertura por medios y saldo inicial.
  2. Cargar movimientos con categoría y comprobante.
  3. Conciliar diferencias antes de cerrar.
  4. Guardar cierre diario para auditoría.
Captura real pendiente: cierre de caja (resumen por medios).

7) Despostada y rendimiento

  1. Ingresar peso inicial por especie.
  2. Distribuir en cortes resultantes.
  3. Controlar merma y rendimiento final.
  4. Impactar automáticamente en stock.
Captura real pendiente: módulo Despostada.

8) Sucursales y transferencias

  1. Verificar sucursal activa antes de operar.
  2. Transferir stock con trazabilidad.
  3. Evitar mezcla de datos entre sucursales.
  4. Controlar recepción en destino.
Captura real pendiente: módulo Sucursales/Transferencias.

9) Logística y reparto (PRO)

  1. Asignar pedidos a repartidores.
  2. Seguir estado de entregas.
  3. Registrar incidencias.
  4. Cerrar circuito con confirmación.
Captura real pendiente: panel Logística.

10) Seguridad y mantenimiento

  1. Administrar usuarios, roles y permisos.
  2. Configurar PIN/código de autorización.
  3. Realizar respaldos periódicos.
  4. Mantener Bridge y app actualizados.
Captura real pendiente: módulo Seguridad.
