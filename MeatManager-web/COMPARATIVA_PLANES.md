# Propuesta Comercial: Sistema MeatManager 🥩

MeatManager es una solución integral diseñada para digitalizar y optimizar la gestión operativa y administrativa de carnicerías. Ofrecemos dos esquemas adaptados a la madurez de cada negocio: el plan **LIGHT** para una gestión eficiente del día a día, y el plan **PRO** para un control total de la rentabilidad.

> **⚠️ IMPORTANTE - SISTEMA PARA CONTROL INTERNO:** Este software está diseñado exclusivamente para la **gestión administrativa privada, auditoría de inventario y control de rentabilidad**. No posee conexión con ARCA (Ex-AFIP) y no emite comprobantes fiscales. Es una herramienta de gestión para el dueño del negocio.

---

## � Plan LIGHT: Gestión Digital Eficiente
*Ideal para negocios que buscan digitalizar su mostrador y controlar el dinero.*

La versión base de MeatManager no es una versión "de prueba", es un sistema de gestión completo que incluye:

*   **Arquitectura Híbrida (Local Primero):** El sistema funciona al 100% en la PC local. No depende de que internet funcione para vender. Si se corta el WiFi, usted sigue cobrando; cuando vuelve el internet, los datos se sincronizan solos.
*   **Punto de Venta (POS) Profesional:** Interfaz ultra-rápida para registrar ventas por kilo o unidad. Registro manual de peso.
*   **Gestión de Medios de Pago:** Configuración dinámica de Efectivo, Tarjetas (con recargos), Transferencias y Billeteras Virtuales (Mercado Pago, Modo, etc.).
*   **Control de Stock de Cortes:** Inventario en tiempo real de todos tus productos (Asado, Vacío, Pollo, etc.) con alertas de faltantes.
*   **Ficha de Clientes y Fiados (Cta. Cte.):** Base de datos de clientes con historial de compras y control riguroso de saldos pendientes (deudas).
*   **Menú Digital para WhatsApp:** Generación automática de catálogo de precios actualizado para enviar a tus clientes por WhatsApp con un solo clic.
*   **Seguridad y Accesos:** Sistema de permisos para que los empleados solo vean lo que tú decidas.
*   **Base de Datos Local:** El sistema funciona al 100% sin internet, garantizando que nunca dejes de vender.

## 👑 Plan PRO: Inteligencia y Rentabilidad Máxima
*Ideal para dueños que quieren maximizar márgenes y profesionalizar la logística.*

Además de todo lo incluido en el plan LIGHT, el Modo PRO desbloquea el motor de inteligencia de MeatManager:

### 1. Trazabilidad y Rendimiento (Rindes)
Deja de adivinar si una media res fue buena o mala. El sistema permite vincular cada corte vendido a un animal específico, calculando el **porcentaje de rinde real** (carne limpia vs. hueso/grasa).

### 2. Monitor de Costos Reales
El sistema calcula el costo de cada kilo de carne después de la despostada. Si el precio del ganado sube o el rinde baja, el sistema te avisa inmediatamente para que ajustes tus precios y no pierdas dinero.

### 3. Gestión Profesional de Proveedores
*   **Ranking de Calidad:** Identifica automáticamente qué proveedor te entrega la mercadería con mejor rinde. 
*   **Estado de Cuenta:** Control detallado de facturas pagadas y pendientes con tus proveedores de carne.

### 4. Flujo Sanitizado (Mostrador -> Caja)
Optimice la higiene y la seguridad. El carnicero puede cargar los pedidos desde una tablet o PC en el mostrador (sin tocar dinero) y el cliente abona su ticket en la Caja Central. Este flujo unifica los cobros y asegura que el personal que manipula alimentos no manipule efectivo.

### 5. Conexión Automatizada con Balanzas
Elimine errores manuales y gane velocidad. El sistema se conecta vía Serial/USB a su balanza (Systel, Kretz, Moretti, etc.) para capturar el peso real directamente en el ticket, sin intervención del operador.

### 6. Gestión Multi-Sucursal y Respaldo en la Nube
A diferencia de los sistemas de red local tradicionales, si la PC principal falla, su información está protegida. Al contar con el servicio de nube activo, su base de datos se sincroniza automáticamente. Esto permite al dueño monitorear ventas y cambiar precios desde su celular sin necesidad de estar físicamente en el local.
